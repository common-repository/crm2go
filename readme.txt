Muestra un formulario de contacto conectado a la base de datos de tu sistema de CRM. Requiere registro de una cuenta gratuita (https://www.crm2go.net)

== Description ==
Permite mostrar un formulario para recibir información de contactos desde tu sitio WordPress en la base de datos de CRM 2go. Para poder utilizar este plugin es necesario el registro de una cuenta gratuita en https://www.crm2go.net.

Para más información y obtener soporte ingresa a https://www.crm2go.net/kb/integrar-tu-sitio-wordpress/.

Visita nuestros términos de servicio y política de privacidad si tienes alguna duda:
https://www.crm2go.net/terminos
https://www.crm2go.net/privacidad


== Installation ==
1. Descarga y descomprime el plugin
2. Sube los archivos a la carpeta \'/wp-content/plugins/crm2go\'
3. Activa el plugin desde \'Plugins\' en tu Escritorio de Wordpress
4. Inserta el código [crm2go_contacto slang=\"SLANG\"] donde quieras mostrar tu formulario
(*) Reemplaza SLANG por el correspondiente a tu sistema.

== Frequently Asked Questions ==
= Recibo un error que dice \"El sistema no existe\" =
Si no tienes una cuenta de CRM 2go debes registrarte, ingresa a https://www.crm2go.net/ para hacerlo. Si ya tienes una cuenta es probable que estés ingresando mal el slang, recuerda es la parte final de la dirección de tu sistema, justo después de https://app.crm2go.net/SLANG.
